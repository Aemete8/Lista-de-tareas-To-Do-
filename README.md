# ✅ To-Do App

Una aplicación sencilla de lista de tareas construida con HTML, CSS y JavaScript vanilla.

## 🚀 Cómo usar

1. Clona el repositorio:
   git clone https://github.com/tuusuario/todo-app.git

2. Abre index.html en tu navegador.

## 📁 Estructura

lista-tareas/ <br>

│   └── app.js <br>
├── css/ <br>
│   └── style.css <br>
├── README.md <br>
└── .gitignore <br>

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)

## 📌 Funcionalidades

- Agregar tareas
- Marcar tareas como completadas
- Eliminar tareas
